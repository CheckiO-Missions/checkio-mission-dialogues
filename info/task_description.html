Modern world is full of means of communication - social networks, messengers, phone calls, SMS etc. But sometimes every communication channel show own problem and you think at moments like that: "I should create my own messenger - without any promblems like this one".
In this mission you have such opportunity.
Your task is to create a NewMessenger class which could help two person (Person1 and Person2) to make a conversation. This class should have method connect_person() for adding person in the dialogue and show_dialogue() to print out the dialogue as a multiline string. Every single line shold looks like:
Person1 (or Person2) said: text of the message
Interlocutors should have a method send() for sending a messages to the dialogue.
In this mission you could use the Mediator design pattern.

Example: 

messenger = NewMessenger()
karl = Person1(messenger)
peter = Person2(messenger)
messenger.connect_person(karl)
messenger.connect_person(peter)
karl.send("Hi! What's new?")
peter.send('Hello. Could we speak later about it?')

Person2 said: Hello. Could we speak later about it?'''


Input: Interlocutors and the text of the messages.

Output: dialogue (multiline string).

How it is used: For creating the connection and sending/receiving the information.

Precondition: 2 interlocutors.
