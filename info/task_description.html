Modern world is full of means of communication - social networks, messengers, phone calls, SMS etc. But sometimes every communication channel show own problem and you think at moments like that: "I should create my own messenger - without any promblems like this one".
In this mission you have such opportunity.
Your task is to create a Chat class which could help Human(name) and Robot(serial_number) to make a conversation. This class should have a few methods:
connect_human() - connect human to the chat.
connect_robot() - connect robot to the chat.
show_human_dialogue() - shows the dialogue as it looks to the human - as simple text.
show_robot_dialogue() - shows the dialogue as it looks to the robot - as the set of the 1 and 0. To simplify the task, just replace every vowel ('aeiouAEIOU') with "0", and the rest symbols (consonants and special signs like ",", "!" etc.) with "1".
Dialogue for human shoud be shown like this:
(human name) said: message text
(robot serial number): message text
For robot:
(human name) said: 11100100011
(robot serial number) said: 100011100101
Interlocutors should have a method send() for sending a messages to the dialogue.
In this mission you could use the Mediator design pattern.

Example: 

chat = Chat()
karl = Human("Karl")
bot = Robot("R2D2")
chat.connect_human(karl)
chat.connect_robot(bot)
karl.send("Hi! What's new?")
bot.send("Hello. Could we speak later about it?")

chat.show_robot_dialogue() == """Karl said: 101111011111011
R2D2 said: 10110111010111100111101110011101011010011011"""


Input: Interlocutors and the text of the messages.

Output: dialogue (multiline string).

How it is used: For creating the connection and sending/receiving the information.

Precondition: 2 interlocutors.
